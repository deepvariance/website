# Deep Variance — Marketing Site

Angular 20 + TailwindCSS landing site for [deepvariance.com](https://deepvariance.com).

## Development

```bash
npm install
ng serve        # http://localhost:4200
```

## Build & Deploy

```bash
ng build        # production build → dist/
```

Deployed automatically to **Vercel** on push to `main`.
